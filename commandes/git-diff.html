Git Diff

Cette commande va montrer les modifications de fichier après un commit, et cela par raport aux commits précédent, de manière très précise.

Syntaxe :

$ git diff

--- a/commandes/git-diff.html
+++ b/commandes/git-diff.html

montre les délétions exactes et en dessous les ajouts exacts

git diff --stat

Il s'agit de la commande qui précisera les changements du dernier commit effectué, mais en beaucoup moins développé par rapport à git diff, ne donnant que le nombre d'insertions et de déletions.

Syntaxe :

$git diff --stat
commandes/git-diff.html | 8 +++++---
1 file changed, 5 insertions(+), 3 deletions(-)

git diff --cached

Il s'agit de la commande qui permettra de comparer deux commits entre eux, donc d'observer les changements de l'un à l'autre.

Syntaxe :

$git diff --cached
